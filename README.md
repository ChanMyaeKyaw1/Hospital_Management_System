# Hospital Management System

* Developed a comprehensive healthcare management system to streamline patient care, hospital administration, and operational workflows.
* Designed a relational database structure to manage patients, doctors, appointments, treatments, billing records, rooms, staff, and pharmacy inventories.
* Established relationships between multiple entities to ensure data consistency, integrity, and efficient information retrieval.
* Applied database transaction management concepts to support reliable handling of appointments, billing processes, and medical records.
* Modeled the system using UML diagrams, including Use Case, Class, Object, Sequence, Collaboration, and State Diagrams.
* Analyzed real-world hospital workflows to define system requirements and user interactions across different roles.
* Implemented role-based functionalities for administrators, doctors, nurses, receptionists, pharmacists, and patients.
* Designed workflows for appointment scheduling, treatment management, billing operations, and inventory tracking.
* Focused on reducing manual processes, improving data accessibility, and supporting efficient healthcare service delivery.

### Highlights

* Centralized management of patient records and medical history
* Appointment scheduling and doctor availability tracking
* Billing, payment, and treatment management
* Pharmacy and medicine inventory monitoring
* Role-based access control for different hospital staff
* UML-based system analysis and architecture design
* Scalable and secure healthcare workflow management

### Technologies & Concepts

MySQL, UML Modeling, Database Design, Software Architecture
